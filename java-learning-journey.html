💡 Biggest learning so far: Debugging real problems helped me understand Java much better than just reading theory. When you hit a race condition, a NullPointerException in a lambda, or a HashMap lookup that returns null — trace it, fix it, then explain why it happened. That's exactly what interviewers ask at 3–5 YOE.
